# Instagram Copy Prompt

You are writing a single Instagram post (caption + image direction) for an Art of Living {{eventType}} in {{region}}.
Event date: {{eventDate}}.

## Your Reference
Follow every principle in docs/COPY-PRINCIPLES.md. This is mandatory.

## Messaging Arc: Feel → See → Believe → Act

1. **Feel (1-2 lines):** Open with a small, specific moment the reader recognizes. The first line must create curiosity — Instagram truncates after ~125 characters. One thought per line. Use line breaks generously.
2. **See (the quote):** A short Gurudev quote (under 15 words preferred). Own line with attribution.
3. **Believe + Act:** Name the program, dates, location. "Link in bio" — soft, not pushy.

## Constraints
- Caption: 80-120 words total
- First line under 125 characters (before truncation)
- One Gurudev quote, verbatim with attribution
- 3-5 relevant hashtags at the end only
- Line breaks between thoughts

## Image Direction
Include a brief image direction (2-3 sentences):
- Photography-style, not illustration
- Real and natural, not staged or produced
- Local feel when possible (Pacific Northwest for WA, etc.)
- No text overlay on the image
- Show life, not a retreat center
- Warm, inviting, contemplative — a moment someone would want to step into

## What Never to Do
- Never start with the quote
- Never use emojis in the caption
- Never use "Link in bio!" with exclamation — use "Link in bio" or "Link in bio if this speaks to you"
- Never use stock photo language ("diverse group of people smiling")
- Never produce more than one post unless specifically asked

## Output
Return the image direction first, then the caption. No meta-commentary.
