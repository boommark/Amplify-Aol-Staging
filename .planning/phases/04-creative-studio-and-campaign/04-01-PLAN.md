  <action>
    Create `lib/pipeline/ad-creative-image.ts` by cloning the pattern from `lib/pipeline/quote-image.ts`. This file exports 4 functions:

    1. **`buildAdCreativePrompt(params)`:** Fetches the prompt template from Supabase `prompts` table using key `image.ad-creative.{channel}.{flavor}`. Uses `renderPrompt()` from `lib/prompts/renderer.ts` to inject variables: `workshopTheme`, `region`, `channelCopy`, `brandPalette` (hardcoded to `#3D8BE8, #E47D6C, #ED994E, #F7C250`). Returns the rendered prompt string.

    2. **`generateAdCreativeImage(params)`:** Direct fetch to `https://generativelanguage.googleapis.com/v1beta/models/gemini-2.5-flash-image:generateContent?key=${process.env.GOOGLE_GENERATIVE_AI_API_KEY}`. Body: `{ contents: [{ parts: [{ text: prompt }] }], generationConfig: { responseModalities: ['TEXT', 'IMAGE'] } }`. Extract base64 from `data.candidates[0].content.parts[N].inlineData` where `mimeType` starts with `image/`. Upload to S3 at key `{userId}/campaigns/{campaignId}/ad-creatives/{channel}-{flavor}.png` using `generatePresignedUploadUrl`. Save to `campaign_assets` via `saveAsset({ campaignId, assetType: 'ad_creative', channel, s3Key, s3Url: publicUrl, metadata: { flavor, promptUsed: prompt, channel, generatedAt: new Date().toISOString() } })`. Return `{ imageUrl: publicUrl, assetId: asset.id }` or null on failure.

    3. **`generateAllAdCreatives(params)`:** Takes `campaignId`, `userId`, `flavor`, `copyResults` (array of `{ channel, content }`), `workshopTheme`, `region`, and optional `onChannelComplete` callback. Uses `Promise.allSettled` across `['instagram', 'facebook', 'whatsapp', 'flyer']`. For each channel: calls `buildAdCreativePrompt` then `generateAdCreativeImage`, then `onChannelComplete(channel, result)`. Returns `Record<string, { imageUrl: string | null; assetId: string | null }>`.

    4. **`refineChannelImage(params)`:** Takes `assetId`, `campaignId`, `channel`, `instruction`, `userId`. Loads existing asset via `getAssetsForCampaign` filtered by assetId. Reads `metadata.promptUsed` from the asset. Builds refinement prompt: `previousPrompt + "\n\nRefinement instruction: " + instruction`. Calls `generateAdCreativeImage` with the combined prompt (same channel, same flavor from metadata). Updates the existing asset via `updateAssetContent(assetId, { s3Url: newUrl, metadata: { ...existingMeta, promptUsed: combinedPrompt, refinedAt: new Date().toISOString() } })`. Returns `{ channel, imageUrl, assetId }`.

    **Important:** Do NOT use `generateImage` from AI SDK. Use direct `fetch` to the Gemini API endpoint. The `TASK_MODEL_MAP` entries are for reference/logging only. This follows the pattern validated in `sample-creatives/generate.mjs`.
