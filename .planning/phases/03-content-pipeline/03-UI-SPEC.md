## Interaction Contracts

### Progressive Research Loading

- 7 ResearchCards stream in one by one as each Perplexity query completes.
- Each card enters with `animate-slideUp` (0.5s ease-out) — not all at once.
- Loading placeholder: `SkeletonPart type="card"` for each card slot before data arrives.
- Cards are clickable to expand from first render; user need not wait for all 7.
- After all 7 load: ActionChips appear ("Add notes", "Scan competitors", "Continue to wisdom") with `animate-slideUp` 200ms delay.

### Competitor Scan

- Triggered only via "Scan competitors" action chip — never auto-triggered.
- Shows a single loading skeleton card with label "Scanning competitor content..." during fetch.
- Results render as a grouped ResearchCard with topic "Competitor Insights" and findings per brand.

### Gurudev Wisdom Cards

- 3 QuoteCards render horizontally in ImageCarousel-style container after research stage.
- Each card has a format tab: Short / Medium / Long — clicking swaps the displayed text (no re-query).
- If Ask Gurudev API times out: show a single QuoteCard with fallback copy ("Wisdom unavailable — continuing with your research context") and a "Try again" action chip.
- "manual" category responses: prefix with "This text is not from Gurudev, but written by the AskGurudev Team" displayed in `text-xs text-slate-500 italic` above the quote.
- If "suicide" meta flag detected: do not render a QuoteCard. Insert an AI text bubble: "I noticed this topic may touch on emotional crisis. Here are some support resources: [crisis helpline details]."

### Channel Selector

- Appears as a ChannelSelector component after wisdom stage action chip "Continue to copy" is clicked.
- Default state: Email, WhatsApp, Instagram, Facebook, Flyer — all selected.
- "+ Custom" opens a small shadcn `Input` inline (not a modal) prompting "Channel name (e.g. TikTok, LinkedIn)". Pressing Enter adds a new chip in selected state.
- "Generate Copy" CTA button: peach `#E47D6C` background, white text, `rounded-full`, Work Sans 16px weight 600, 48px height, full-width on mobile, 200px fixed width on desktop. Disabled state: `opacity-50 cursor-not-allowed`.

### Copy Generation

- Each channel's CopyBlock renders as it completes (not batch).
- Order: Email first, then WhatsApp, Instagram, Facebook, Flyer, custom channels.
- Each block enters with `animate-slideLeft` 0.5s ease-out.
- Each CopyBlock gets a "Refine" action chip below it: "Make shorter", "More formal", "Translate". Clicking sends a targeted refinement — only that channel's block re-renders (skeleton → updated content).
- Copy-to-clipboard button (`aria-label="Copy to clipboard"`, Lucide `Copy` icon) on each block: `slate-400` default, `#3D8BE8` on hover, shows Lucide `Check` icon for 1500ms on success.

### Research Reuse

- If matching region research found: ResearchReusePrompt appears before any Perplexity queries fire.
- "Use existing" skips queries, renders existing ResearchCards from DB.
- "Run fresh" proceeds with 7 Perplexity queries as normal.

### Streaming + Loading

- Skeleton loaders for all async parts with wait time > 300ms (per design system rule).
- SkeletonPart `type="card"` for research cards and quote cards.
- SkeletonPart `type="image"` for quote background images.
- No blank screens at any stage — user always sees either content or skeleton.
- StageProgressBar updates step state at each stage transition.

## Accessibility

- All expandable ResearchCards: `role="button"` + `aria-expanded` (already implemented in code).
- ChannelSelector chips: `role="checkbox"` + `aria-checked` per chip.
- QuoteCard format tabs: `role="tablist"` + `role="tab"` + `aria-selected`.
- StageProgressBar: `role="list"` + `aria-current="step"` for active step.
- Icon-only buttons require explicit `aria-label` values: copy-to-clipboard `aria-label="Copy to clipboard"`, download `aria-label="Download quote card"`, stop-stream `aria-label="Stop generating"`.
- Focus ring: visible 2px ring using `ring-2 ring-[#3D8BE8]` on all interactive elements.
- `prefers-reduced-motion`: disable `animate-slideUp`, `animate-slideLeft`, `accordion-down/up` animations. Show content immediately without transition.
- Minimum contrast 4.5:1 for all text/background combinations.
- Touch targets minimum 44x44px for all clickable elements.
